Recursos

Moderacion!

Un sistema de moderacion simple pero eficas, que puede ser muy util en tu servidor, modera a tus usuarios y evita que rompan las reglas de tu servidor haz de tu server un lugar mas seguro mas informacion

Divertido!

Utiliza los comandos mas divertidos y haz que tus usuarios pasen mucho tiempo jugando los minijuegos o puedes hacer competencias muy buenas! al igual que hacer algunos sorteos

Funciones!

Off topic tiene un sistema de configuracion muy avanzado como el sistema de contar o el sistema de niveles que puedes configurar y perzonalizar

Bienvenidas

crea un sistema de bienvenidas para tu servidor, un sistema que puedes perzonalizar y hacer que tu server sea el mejor de discord!

Contar

Crea un sistema de contar para que tus usuarios se entretengar y intenten llegar muy lejos, puedes hacer competencias muy buenas!

Auto respuestas

Crea un sistema de autorespuestas para que los usuarios puedan charlar con el bot o crear tus propios comandos perzonalizados y hacer de tu servidor un lugar unico!
